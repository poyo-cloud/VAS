# VAS

iPad 向けの静的 Web アプリです。本実験の流れに合わせ、コントロール 1 回と条件 1〜4 の計 5 回の主観評価を実施し、端末内保存と CSV 出力を行います。

## できること

- 最初に `患者ID`、`検査実施日`、`パターン` を入力
- パターン A〜H は条件 1〜4 の提示順に対応
- 同一ブラウザでは初回だけ 10cm 校正を要求
- 各評価回を `VAS` と `スタンフォード眠気尺度・reaction time` の 2 ページに分けて入力
- VAS は 6 項目
  - 香りをどのくらい感じたか
  - 香りはどのくらい好きだったか
  - 香りはどのくらいツンとしたか
  - いまの心地よさはどのくらいか
  - いまの疲れはどのくらいか
  - いまの集中力はどのくらいか
- スタンフォード眠気尺度 日本語版 1〜7 と `reaction time` 視覚 3 回分・聴覚 3 回分を入力
- 端末内に履歴保存
- 単回または履歴まとめて `CSV` 出力

## パターン

- パターンA: 3 → 1 → 4 → 2
- パターンB: 2 → 4 → 1 → 3
- パターンC: 4 → 2 → 3 → 1
- パターンD: 1 → 3 → 2 → 4
- パターンE: 3 → 4 → 2 → 1
- パターンF: 2 → 1 → 4 → 3
- パターンG: 4 → 3 → 1 → 2
- パターンH: 1 → 4 → 3 → 2

## CSV 出力

- 1 人の被験者につき 1 行で出力します
- CSV は `SubjectNo` `Pattern` に続いて、コントロールと条件 1〜4 の列を出力します
- VAS 値は `0.0` 〜 `10.0` の cm 値で小数点第一位まで出力します
- スタンフォード眠気尺度は `sleepiness` として 1〜7 の選択値を出力します
- `reaction time` は視覚 3 回・聴覚 3 回それぞれの平均を出力します
- 1 件ずつの出力ファイル名は `VAS_<患者ID>.csv` です
