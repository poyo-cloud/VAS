# Research VAS

iPad・タブレット向けの静的Webアプリです。研究分野を限定せず、ユーザーが条件、VAS項目、提示順を自由に設定できます。

## 起動

`index.html` をWebサーバーから開きます。データは利用中のブラウザの `localStorage` に保存されます。

```sh
cd VAS
python3 -m http.server 8871
```

ブラウザで `http://127.0.0.1:8871/` を開いてください。

## 主な機能

- 複数プロジェクトの作成、編集、複製、削除
- プロジェクト一覧から直接実験を開始
- プロジェクト、条件、VAS項目、パターンをタッチ操作で並べ替え
- 条件・刺激を1つの編集画面で修正、追加、削除、並べ替え
- 既存・新規のVAS項目を1つの編集画面で修正、追加、削除、並べ替え
- VASは10cm固定（左端0 mm、右端100 mm）
- 条件提示パターンの手動作成と並べ替え
- seedを保存する再現可能なランダムパターン生成
- 同一提示順を除外し、既存パターンを保持したまま追加
- 被験者ID、セッションID、パターンを指定した実験
- 同一条件の全VAS項目を1画面に表示
- 条件名・条件説明を回答者へ表示しない盲検化設定
- 10cmの基準線を指やペンのフリーハンド線で横切るVAS
- 実施履歴の端末内保存
- 7シートを含むXLSX出力

## XLSXシート

- `project`
- `conditions`
- `vas_items`
- `pattern_master`
- `assignment_log`
- `raw_responses`
- `wide_by_condition`

`raw_responses` は実際の提示順を保持します。`wide_by_condition` は条件とVAS項目から動的に列名を作り、条件順を統一します。

XLSX生成には同梱した SheetJS Community Edition を使うため、出力時のインターネット接続は不要です。

## データについて

ブラウザのサイトデータを消去すると保存データも消えます。研究運用では実験終了ごとにXLSXを出力し、別媒体へ保管してください。クラウド同期、ログイン、暗号化バックアップはMVPの対象外です。
